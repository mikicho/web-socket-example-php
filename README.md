Example of PHP websockets based on [this guide](http://phppot.com/php/simple-php-chat-using-websocket/).   
Steps for run: 
1. run `php -q .\socket.php`
2. run `php -S localhost:8000`
3. go to `localhost:8000` in your favorite browser
